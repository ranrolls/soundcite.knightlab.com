Finally embedding audio is easy

Audio clips force uncomfortable choices: either reads or listen but not both. Until now.

Audio is powerful device that can add emotion or context to a story. SoundCite is simple-to-use tool that lets you add audio to your story. The audio is not isolated, it plays right under the text you choose. SoundCite is currently in an alpha, pre-release stage.

Creating Your Clip

1

Find your SoundCloud sound

Find the sound you want to clip from SoundCloud and copy its URL.

Paste it into the box on the clip creation page.

2

Set start and end points

Play through the sound and find the section you want to clip. At the start of your clip, click the start button to set the start point. At the end of your clip, click the end button to set the end point.

3

Paste your text

Paste the text you want to link to your clip in the third box of the Clip Creator.

4

Create your clip

Click create clip to see a preview of your clip. Repeat steps 3-5 to create more clips from this sound. Use the delete button to remove clips that you do not want.

Embedding Your Clip

1

Embed the header code

In the left column of the Clip Creator, copy the header embed code and paste it in the top of your post in your CMS. You only need to do this once per post.

2

Embed each clip

For each clip created, copy its embed code and paste it where the text belongs in your post.

Questions & Answers

Do I need to know <code> to use SoundCite?
Absolutely not. SoundCite is friction-free. If you can record audio, upload it to soundcloud.com you are ready to use SoundCite.
Can I customize what SoundCite looks like?
SoundCite is designed to work with virtually any site or theme design. But if you want to change the size, color or shading you'll be able to modify the stylesheets to do so ... soon.
Can I use an audio source other than SoundCloud?
We're working on this too. Hopefully we can start expanding the potential sources soon.
Can I help build new features for SoundCite?
Absolutely. Fork the project and get involved. Send us feedback. Help us make SoundCite the tool you need and want it to be.

About SoundCite

SoundCite is one of the Northwestern University's Knight Lab tool's for content creators. It was created by Medill student Tyler Fisher and Medill Asst. Professor Jeremy Gilbert.
